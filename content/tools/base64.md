### 什么是 Base64 编码？

Base64 是一种基于 64 个可打印字符来表示二进制数据的表示方法。它常用于在通常处理文本数据的场合传输或存储二进制数据。

简单来说，它可以把一张**图片**变成一长串**字符**，或者把一段**文字**变成另一种**编码格式**。

### 常见应用场景

-   **图片嵌入网页**：将小图标转换为 Base64 字符串直接写在 CSS 或 HTML 中，减少 HTTP 请求，提高网页加载速度。
-   **数据传输**：在 HTTP 请求中传输二进制数据时，Base64 可以防止特殊字符导致的解析错误。
-   **简单的加密**：虽然 Base64 不是加密算法，但常被用于隐藏明文信息，防止被肉眼直接识别。

### 常见问题 (FAQ)

#### Base64 编码安全吗？
Base64 **不是加密**。它只是一种编码方式，任何人都可以轻松地将 Base64 解码回原始内容。因此，**不要**使用 Base64 来保护密码或敏感信息。

#### 为什么 Base64 编码后体积会变大？
Base64 编码后的数据体积通常会比原始数据大约 **33%**。这是因为它用 4 个字符来表示 3 个字节的数据。
