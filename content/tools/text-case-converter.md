### 什么是文本大小写转换？

文本大小写转换工具可以帮助你快速改变字符串的格式。这对于编程命名（如将 `user name` 变为 `userName`）或文案排版（如将 `hello world` 变为 `Hello World`）非常有用。

### 支持的格式

-   **全大写 (UPPERCASE)**：所有字母都大写。常用于常量定义或强调语气。
-   **全小写 (lowercase)**：所有字母都小写。常用于 URL 或文件名。
-   **首字母大写 (Title Case)**：每个单词的首字母大写。常用于文章标题。
-   **小驼峰 (camelCase)**：第一个单词小写，后续单词首字母大写。常用于 Java, JavaScript 变量命名。
-   **下划线 (snake_case)**：单词之间用下划线连接，通常全小写。常用于 Python 变量命名。
-   **短横线 (kebab-case)**：单词之间用短横线连接。常用于 CSS 类名或 URL。

### 常见应用场景

-   **编程变量命名**：快速将复制的文本转换为符合代码规范的变量名。
-   **CSS 类名编写**：快速生成 kebab-case 格式的类名。
-   **标题规范化**：统一文章标题的大小写格式。
