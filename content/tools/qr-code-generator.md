### 什么是 QR Code 二维码生成器?

QR Code 生成器是一款完全免费、完全本地化的在线工具,用于将文本、网址、WiFi 网络、联系人(vCard)等内容转换为标准 QR 二维码。所有计算都在你的浏览器内完成,**不上传任何数据到服务器**,保护隐私。

### 这个工具能做什么?

* **文本模式**:任意字符串转二维码(纯文本、数字、JSON)
* **网址模式**:把长网址变成可扫码的短链,适合印刷在名片、海报、商品包装上
* **WiFi 模式**:把 WiFi 名称 + 密码打包成二维码,客人扫码自动连网(无需手动输入密码)
* **邮箱模式**:一键生成"邮件二维码",扫码自动打开邮件 App 并填好收件人 / 主题
* **电话模式**:扫码直接拨打电话
* **联系人模式**:生成 vCard 二维码,扫码自动导入联系人(姓名、电话、邮箱、公司)
* **SMS 模式**:扫码自动打开短信 App,预填收件人和内容

### 如何使用?

1. **选择内容类型**:在顶部 Tab 选择要编码的内容类型(文本 / 网址 / WiFi / 邮箱 / 电话 / 联系人 / SMS)

### 常见问题(FAQ)

**Q:二维码生成的数据会上传到服务器吗?**
A:不会。所有计算都在你的浏览器内完成,完全本地处理,断网也能用。

**Q:WiFi 二维码怎么用?客人怎么连?**
A:WiFi 二维码包含 SSID、密码和加密方式。客人用微信 / 相机的扫一扫功能扫码,会提示"加入网络 XXX",一键连接,无需手动输入密码。

**Q:为什么我的二维码扫不出来?**
A:常见原因:1) 内容太长(URL 超过 200 字符);2) 颜色对比度不够(浅色前景 + 浅色背景);3) 容错率太低(L 等级);4) 二维码尺寸太小(打印小于 2cm × 2cm 难扫)。

**Q:PNG 和 SVG 有什么区别?我应该用哪个?**
A:PNG 是位图,适合屏幕显示、社交媒体分享;SVG 是矢量图,无限放大不失真,适合印刷品、海报、名片。**印刷请用 SVG**。

**Q:可以批量生成二维码吗?**
A:当前版本一次生成一个。如需批量,可以先用本工具生成单个测试,确认效果后再用 Python/Node 脚本批量生成(基于相同 qrcode.js 库)。

**Q:支持中文内容吗?**
A:支持。UTF-8 编码自动处理,中文 / 日文 / 韩文均可。

**Q:二维码有有效期限吗?**
A:静态二维码(本工具生成)**永久有效**。但如果内容是 URL,URL 链接的网站可能失效;如果是 WiFi,密码改了二维码就失效。

**Q:可以加 Logo 到二维码中间吗?**
A:当前版本未支持。技术上可行(QR 容错率 H 可覆盖 30% 区域),需要付费版。
