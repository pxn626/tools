Built on the open-source pdf-lib library, processes 100% in your browser — works offline after first load.

## How to Use

Three clicks from a pile of PDFs to a single file. Here's how.

### To Merge PDFs

1. **Drop your PDF files into the upload area** — or click anywhere in the box to open the file picker. The dropzone accepts any number of files; add 2, 20, or 200 PDFs at once.
2. **Drag the file cards to reorder them.** The first card becomes page 1 of the merged PDF, the second card becomes the next chunk, and so on. You can also remove any file before merging.
3. **Click Merge.** The processing happens in your browser using pdf-lib — no upload, no waiting on a remote server. When the progress bar finishes, click Download to save the merged PDF to your device.

### To Split a PDF

1. **Drop a single PDF into the upload area.** The split tool works on one file at a time — if you need to split multiple PDFs, run the split once per file.
2. **Choose your split mode.** Type a range like `1-3, 5, 7-9` to extract those pages into separate files, or enter a single number to extract just that one page. Whitespace is tolerated.
3. **Click Split.** The tool produces one PDF per range (or a single new PDF containing only the extracted pages). If you get more than one file, they're packaged into a ZIP for one-click download.

## Privacy & Security

Most free PDF tools upload your documents to a remote server, process them there, and stream the result back. That means your contracts, tax returns, medical records, and personal letters pass through infrastructure you can't audit — and the provider can log, store, or scan them along the way.

webpenson is different. This tool is built on **pdf-lib**, an open-source JavaScript library (MIT licensed). When you click **Merge** or **Split**, the code runs in *your browser tab*. Your PDFs are read by JavaScript, processed in memory, and offered back to you as a download — without ever making a network request that carries the file content.

You can verify this yourself: open **DevTools → Network** (press F12 in Chrome / Edge, or Cmd+Option+I on macOS Safari) and try a merge. You'll see zero upload requests for your PDF data. For maximum privacy, disconnect from the internet after the page loads — the tool keeps working offline.

## FAQ

### Is this PDF merger really free?

Yes — fully free, no trial period, no premium tier, and no daily limit on how many PDFs you can merge or split. We don't run ads, we don't watermark the output, and we don't sell your data. This is a free utility from webpenson.

### Are my files uploaded to a server?

No. All processing happens **100% client-side**, in your browser, using the open-source pdf-lib JavaScript library. Your PDF never makes a network request with the file content. You can verify this yourself: open your browser's Network panel (F12 → Network) and watch for any upload while you click Merge or Split — you'll see none.

### Can I split a PDF by page range?

Yes. In the Split tab, enter ranges like `1-3, 5, 7-9` — the tool creates a separate PDF for each range and packages them into a single ZIP for download. You can also split a PDF into individual pages, or extract a custom selection of pages into one new PDF without producing the rest.

### Is there a file size limit?

There's no server-side limit (because there's no server). The browser's memory caps practical size — PDFs over 200 MB may fail on low-RAM devices or take a while to process. For best results, keep individual files under 100 MB. Mobile users should aim for under 30 MB per file.

### Do I need to create an account?

No — no account, no email, no sign-up, no verification step. Open the page, drop your PDFs, download the result. We don't track who you are, and there is no marketing email waiting for you on the other side.

### What PDF formats are supported — and can I use this for legal filings, tax documents, or scanned receipts?

The tool accepts any standard `.pdf` — including PDF/A, scanned PDFs with selectable text, and outputs from common office tools. It's commonly used for court filings, tax returns, and bulk invoice PDFs because files never leave the device. Encrypted or password-protected PDFs aren't yet supported in v1 — open the file in any PDF viewer, remove the password, and re-save.

### Does this work on mobile devices?

Yes — the tool runs in any modern mobile browser (iOS Safari, Android Chrome). Single-file splits and small merges work well. For files over 30 MB on a phone, a desktop browser is more reliable. The tool is fully responsive; nothing to install, no app required.
